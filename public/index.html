The format for a parameter is: <url>/page?parameter=value&additional_parameters=value
That said, here are the parameters:

    Parameters for page download_video
    • id: the ID of the video you want to download. ex: dQw4w9WgXcQ
      • Additional parameters (optional):
      • format: can be mp3 or mp4

    Parameters for page dynamic_page
    • code: the HTML markup of the page. Note: the HTML needs to be minified and URL-encoded.

      • Example:
      https://backend.bg117.repl.co/dynamic_page?code=%3C%21DOCTYPE%20html%3E%3Chtml%3E%3Cbody%3EHello%20World%3C%2Fbody%3E%3C%2Fhtml%3E
