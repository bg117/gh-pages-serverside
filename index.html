The format for a query is: <url>/[subdirectory]?key=value[&additional_keys=value...]
That said, here are the keys:

    Keys for subdirectory download_video
    • id: the ID of the video you want to download. ex: dQw4w9WgXcQ
      • Additional keys (optional):
      • format: can be mp3 or mp4

    Keys for subdirectory dynamic_page
    • code: the HTML markup of the page. Note: the HTML needs to be minified and Base64-encoded.

      • Example:
      https://backend.bg117.repl.co/dynamic_page?code=PCFET0NUWVBFIGh0bWw+PGh0bWw+PGJvZHk+SGVsbG8gV29ybGQ8L2JvZHk+PC9odG1sPg==
